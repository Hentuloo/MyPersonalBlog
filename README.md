# My blog

A project based on principles of atomic design every component is reusable and easy to develop thanks to storybook

## I used to build this site:

- React
- Appolo
- GraphQl and GraphCMS(api)
- Styled-components
- ThemeProvider
- Storybook
- marksy
- jsx-parser

### The site is checked thanks to:

- eslint
- prettier
- husky
- lint-staged
